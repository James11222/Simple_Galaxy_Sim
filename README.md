# Simple Galaxy Simulation
A simple illustration of how one can simulate a galaxy in python. To use this, first install 

`numpy`
`matplotlib`
`celluloid`
`scipy`

These packages allow us to run the `galaxy.py` script. Once all necessary packages are installed, run 

`python galaxy.py` and an animated simulation of the galaxy should appear as `galaxy.gif`. On the left is the simulated positions and on the right is the simulated velocity curve.
